# TokenGate

Access control based on token ownership.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/TokenGate.sol`
- Stacks: `contracts/stacks/contracts/tokengate.clar`

## Setup
```bash
pnpm install
pnpm dev
```
